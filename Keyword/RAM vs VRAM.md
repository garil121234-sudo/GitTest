# RAM vs VRAM
둘 다 데이터를 임시로 저장하는 메모리  
|구 분 |RAM|VRAM
|---|---|---|
사용 주체|CPU				   	 |GPU(그래픽카드)
역할	 |프로그램과 데이터 저장 |드래픽 데이터 저장
위치	 |메인보드의 메모리	 	 |그래픽카드 내부
용량	 |(8,16,32)GB			 |(4,8,12,16)GB

### RAM
RAM은 컴퓨터가 작업할 때 사용하는 작업 공간  
예를 들면  
* 인터넷 브라우저 실행
* 게임 실행
* C# 프로그램 실행
* 문서 작업  
RAM이 부족하면 컴퓨터가 버벅거리거나 느려진다

### VRAM
VRAM은 그래픽카드 전용 메모리  
예를 들면  
* 게임 텍스처
* 3D 모델
* 영상 데이터
* AI 연산 데이터 등을 저장  
VRAM이 부족하면 프레임이 떨어지거나 텍스처 품질이 낮아진다
