# TGA
고품질 이미지를 저장하기 위해 개발된 이미지 파일 형식  
게임 개발과 그래픽 분야에서 많이 사용되며 알파채널(투명도)을 지원하는 것이 큰 특징  

### 특징
- 확장자 : .tga
- 개발 : Truevision
- 압축 방식 : 무압축 또는 RLE 무손실 압축
- 화질 : 손실 없음
- 투명도 : 알파 채널 지원
- 색상 : 8, 16, 24, 32비트 지원

### 장점
- 화질 손실이 없다
- 알파 채널을 지원하여 투명한 이미지를 저장 할 수 있다
- 구조가 단순하여 그래픽 프로그램에서 다루기 쉽다
- 게임 텍스처와 CG작업에 많이 사용된다

### 단점
- PNG보다 파일 크기가 큰 편이다
- 웹 브라우저에서는 거의 지원하지 않는다
- 일반 사용자에게는 활용 빈도가 낮다

### 사용되는 곳
- 게임 개발의 텍스처
- 3D 모델링 및 렌더링
- 컴퓨터 그래픽
- 이미지 편집 프로그램

### TGA 파일 구조
1. 헤더
    - 이미지 크기
    - 색상 깊이
    - 압축 여부
    - 이미지 종류
1. 색상 맵
    - 팔레트 이미지에서 색상 정보 저장
1. 이미지 데이터
    - 실제 픽셀 정보
1. 개발자 영역 및 확장 영역
    - 추가 메타데이터 저장

BMP 와의 차이
항목     |	BMP           |	TGA
---      |---             |---
압축     |	대부분 무압축 |	무압축 또는 RLE 압축
화질     |	손실 없음     |	손실 없음
알파 채널|	거의 지원     | 안 함	지원
파일 크기|	큼            |	BMP보다 작을 수 있음(RLE 사용 시)
사용 분야|	Windows 이미지|	게임, CG, 3D 그래픽

**게임 개발에 많이 사용하는 이유**  
게임에서는 캐릭터, UI, 이펙트 등에 투명한 부분이 필요한 경우가 많다  
TGA는 알파 채널을 저장 할수 있어 배경이 투명한 이미지를 그대로 사용 할수 있기 때문이다  
과거부터 게임 엔진과 그래픽 툴에서 많이 사용되었다
