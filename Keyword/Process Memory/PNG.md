# PNG
- 무손실 압축을 사용하는 이미지 파일 형식
- 원본 이미지의 품질을 유지하면서 파일 크기를 줄일 수 있다
- 웹에서 가장 많이 사용하는 이밎 포맷 중 하나

### 특징
1. 무손실 압축
    - 이미지를 압축해도 화질이 손상되지 않는다
    - 여러 번 저장해도 품질이 유지된다
    - ex) 아이콘 , 로고 , UI이미지
1. 투명 배경 지원(PNG의 가장 큰 장점)
    - 배경을 완전히 투명하게 만들 수 있다
    - 반투명(알파채널)도 지원한다
    - ex) 게임 캐릭터 스프라이트, UI버튼, 로고
1. 알파 채널(RGBA형식을 지원한다)
    - R (Red)
    - G (Green)
    - B (Blue)
    - A (Alpha:투명도)
    - 알파값
        - 0 = 완전 투명
        - 255 = 완전 불투명
1. 다양한 색상 표현(수백만가지의 색상을 표현할 수 있다)
    - 24비트 컬러
    - 32비트(RGBA)
1. 웹에서 많이 사용
    - 아이콘
    - 버튼
    - UI
    - 로고
    - 투명 이미지

### 장,단점
**장점**
- 무손실 압축
- 투명 배경 지원
- 알파 채널 지원
- 화질이 깨지지 않음
- 웹 그래픽에 적합

**단점**
- JPG보다 파일 크기가 큰 편
- 사진 저장에는 비효율적
- 애니메이션은 기본적으로 지원하지 않음(APNG라는 확장형식은 애니메이션 지원가능)

### 사용분야
- 게임 UI
- 캐릭터 스프라이트
- 아이콘
- 로고
- 웹 이미지
- 버튼 이미지
- 투명 배경이 필요한 그래픽
