# WAV
- 비압축(PCM) 오디오 데이터를 저장하는 대표적인 음성 파일 형식
- 음질 손실 없이 원본 소리를 그대로 저장한다
- Microsoft 와 IBM이 공동으로 개발 했다

### 특징
1. 비압축 오디오
    - 대부분 PCM(Pulse Code Modulation) 방식으로 저장된다.
    - 압축하지 않기 때문에 원본 음질을 그대로 유지한다.
    - ex) 녹음 파일, 효과음, 음성 데이터
1. 고음질
    - 음질 손실이 없다
    - 음악 편집이나 녹음 작업에 많이 사용한다
1. 파일 크기가 큼
    - 압축하지 않기 때문에 파일 크기가 크다
    - ex)같은 길이의 음악 기준 MP3 : 5MB , WAV : 40~50Mb 정도
1. 빠른 재생
    - 압축을 해제할 필요가 없어 CPU 부담이 적다
    - 게임에서도 효과음을 재생할 때 자주 사용된다

### PCM
MAV에서 가장 많이 사용하는 저장 방식  
소리의 아날로그 신호를 일정한 시간 간격으로 샘플링하여 디지털 데이터로 저장한다

**구성 요소**
- 샘플링 레이트 : 1초에 몇 번 소리를 측정하는지
- 비트 깊이 : 한 번 측정할 때 몇 비트로 저장하는지
- 채널 : 모노(1채널), 스테레오(2채널)
- ex) 44.1kHz / 16bit / Stereo
    - CD음질

### 장, 단점
**장점**
- 음질 손실 없음
- 편집 하기 쉬움
- 녹음에 적합
- 효과음에 적합
- CPU 부담이 적음

**단점**
- 파일 크기가 매우 큼
- 인터넷 전송에 비효율적
- 저장 공간에 많이 사용

### 사용 분야
- 게임효과음
- 음성 녹음
- 음악 편집
- 영화 음향
- 방송 제작
- 오디오 원본 저장

### 게임 개발에서의 활용
- 효과음 : WAV를 많이 사용
- 배경음악 : MP3 , OGG를 많이 사용
    - 파일 크기가 작아 메모리 사용량을 줄일 수 있기 떄문
