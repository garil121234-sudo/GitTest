# CPU vs GPU

## CPU
컴퓨터의두뇌  
코어 하나하나의 성능이 강력함
#### 게임 할때
- 몬스터 AI 계산
- 물리 엔진 계산
- 게임 로직 처리

## GPU
게임 그래픽을 그리기 위해 만들어짐  
수많은 픽셀을 한번에 계산하기위해서 만들어짐
#### 게임 할때
- 캐릭터 그림
- 그림자
- 빛 효과
- 화면 전체를 그리는 작업

|구 분|CPU|GPU|
|---|---|---|
|코어 수|적 음|많 음|
|특 징|한번에 복잡한 일을 빠르게 처리|단순한일을 엄청 많이 동시에 처리
|주 용도|운영체제, 프로그램 실행|게임 그래픽, AI,영상처리
|대표 제품|인텔 Core i7,AMD Ryzen| NVIDIA RTX,AMD Radeon
