# Process memory 영역
Process memory 영역은 프로그램이 실행될 때 운영체제가 그 프로그램에게 나누어주는 공간  

C# 프로그램으로 예를 들자면  
* Stack  (높은주소)
* Heap
* Data(Bss, Data)
* Code  (낮은주소)  
프로그램을 실행하면 운영체제는 메모리를 여러 영역으로 나누어 준다.  

1. Code영역  
프로그램의 실행 코드가 들어가는 곳  
명령들이 컴파일 되면 기계어가 되어 이 영역에 저장된다.  
- 특징
- 읽기 가능
- 실행 가능
- 수정은 거의 불가능

2. Data 영역  
프로그램이 시작될 때부터 끝날 때까지 존재하는 데이터를 저장

3. Heap 영역  
사용자가 직접 생성하는 객체들이 저장되는 곳  
Stack 에는 참조(주소)가 있고, Heap에는 실제 객체가 있다.

4. Stack 영역
함수가 호출될 때마다 사용하는 공간  
지역변수, 매개변수가 저장되고 함수가 종료되면 사라진다
