# Stack 영역
메서드(함수)가 실행될 때 필요한 정보를 임시로 저장하는 메모리 공간

**Stack에 저장되는 것들**
- 지역 변수
- 매개변수
- 값 형식
- 참조형식의 주소(참조값)

**Stack은 나중에 들어간 것이 먼저 나온다**
```
┌─────┐ ← 마지막에 올림
│ C   │
├─────┤
│ B   │
├─────┤
│ A   │ ← 처음 올림
└─────┘
```
마지막에 올린 C가 꺼낼 때는 제일 먼저 나온다
(LIFO: Last In, First Out)

**지역 변수는 메서드가 끝나면 사용할 수 없다**

**참조형식도 Stack을 사용한다**
```C#
class Person
{
    public int Age;
}

Person p = new Person();
```
여기서 p는 스텍에 있지만, 실제 Person객체는 Heap에 있다
- Stack에는 주소(참조값)
- Heap에는 실제 객체 가 저장된다
